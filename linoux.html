Linux :

linux@user:~$ linux_info

Linux est un système d'exploitation open source, c'est à dire que son code est disponible pour tout le monde. Vous pouvez retrouver le code source du noyau de linux ici.

linux@user:~$ show_dist

Bon, En réalité, Linux n’est pas un système d’exploitation complet à lui seul, mais un noyau. De nombreuses distributions sont construites autour de ce noyau, chacune avec ses outils, son interface et ses particularités :

Ubuntu Linux Mint Arch Linux Red Hat Kali Linux
C'est certainement la distribution la plus connue. Elle est utilisée partout et est très pratique, autant pour un usage quotidien que pour des serveurs. Linux Mint est connue pour sa grande accessibilité. Elle est facile à prendre en main et possède une interface proche de Windows. Arch Linux est une distribution... minimaliste. Elle est réputée pour être compliquée à utiliser et elle repose principalement sur la ligne de commande Red Hat est une distribution commerciale qui est vendu aux entreprise. Elle est stable, sécurisée et utilisée le plus généralement pour gérer des serveurs Kali Linux est une distribution spécialisée dans la cybersécurité, on y retrouve tout les outils pour effectuer des tests d'intrusion

linux@user:~$ show_aventages

Les aventages de Linux :

  • Gratuit
  • Peut convenir a énormément de cas d'usage différent
  • Pas de marketing vicieux (C’est open source donc normalement on n’essayera pas de vous escroquer ou de vendre vos données )
  • Extrêmement personnalisable
  • Stable (En théorie )

Les incovéniant de Linux :

  • Pas très intuitif
  • On peut dire adieu au gaming ( même si c'est en train de pas mal bouger en ce moment )
  • Pas idéal pour les débutants
  • Pas de support " officiel "
